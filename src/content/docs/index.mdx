이 사이트는 **문서 위주**로, 에디터를 만들 때 필요한 IME 관련 지식만 다룹니다.

- 전 세계 IME 구조와 글자를 조합(composition)해 표현하는 방식
- 키보드·입력기마다 다른 글자 조합 알고리즘
- **한글**: 한글 입력기, 모바일 한글 키보드, 조합 방식 상세

본문은 구체적 사실과 동작 결과만 기술합니다.
