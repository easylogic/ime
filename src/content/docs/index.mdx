## 왜 IME를 이해해야 할까요?

텍스트 에디터, 채팅 앱, 폼 입력 등 **텍스트를 다루는 모든 웹 애플리케이션**은 IME와 상호작용합니다.
한글, 일본어, 중국어 같은 언어는 여러 키 입력을 조합해 한 글자를 만들기 때문에,
이 과정을 제대로 이해하지 못하면 **입력 버그, 글자 깨짐, UX 저하**가 발생합니다.
